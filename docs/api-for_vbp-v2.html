VTU BUSINESS PORTAL VERSION 2

Balance API
https://PORTAL_URL/api/v1/balance?format=json&api-token=xxxxxxx
Example:

https://PORTAL_URL/api/v1/balance?format=json&api-token=API_TOKEN&
Responses:
Success
{"status":200,"message":"success","balance":223900.0}
Failed
{'status': 400, 'message':'error', 'details': 'Bad Request Check Your Parameters'}
End of Balance API


Airtime API
https://PORTAL_URL/api/v1/airtime?format=json&api-token=xxxxxxx&network=xxx&phone=xxxxxxxxxx&amount=xxx
Example:

https://PORTAL_URL/api/v1/airtime?format=json&api-token=API_TOKEN&network=MTN&phone=08163524890&amount=1000
Parameters:
newtwork = network code (Identifier)
phone = recipient number to top-up e.g 08163524890
amount = amount to top-up
Responses:
Success
{'status':201, 'message':'success', 'details': 'Transaction Successful'}
Failed
{'status':'403', 'message': 'Update your profile to proceed'}
{'status':205, 'message':'failure', 'details': 'Transaction not Successful'}
{'status': 422, 'message':'error', 'details':'Insufficient Balance'}
{'status': 400, 'message':'error', 'details': 'Bad Request Check Your Parameters'}
End of Airtime API

ELECTRICITY API
Customer Check

https://otextcity.com/app/api/v1/check_electricity?meterNo=[METER_NO]&service=[SERVICE]

Example:
https://otextcity.com/app/api/v1/check_electricity?meterNo=04042404139&service=IKEJA_TOKEN

Parameters:
meterNo = meter Number
service = ElectricityCode

Responses:
Success
{"status":200,"message":"Customer details retrieved successfully","details":{"canVend":true,"address":"ABULE - EGBA BU ABULE","meterNumber":"04042404139","name":"TESTMETER1","minimumAmount":0,"customerAccountType":"PRIME","customerDtNumber":"007903312","vbpMeterNo":"04042404139","vbpService":"IKEJA_TOKEN"}}


Customer Subscription
https://PORTAL_URL/api/v1/electricityPurchase?api-token=API_TOKEN&meterNo=[METER_NO]&service=[SERVICE]&meterType=[METER_TYPE]&amount=[AMOUNT]&customerName=[CUSTOMER_NAME]&customerAddress=[CUSTOMER_ADDRESS]&customerDtNumber=[CUSTOMER_DT_NUMBER]&uniqueCode=[UNIQUE_CODE]

Example:

https://PORTAL_URL/api/v1/electricityPurchase?api-token=API_TOKEN&meterNo=[METER_NO]&service=[SERVICE]&meterType=[METER_TYPE]&amount=[AMOUNT]&customerName=[CUSTOMER_NAME]&customerAddress=[CUSTOMER_ADDRESS]&customerDtNumber=[CUSTOMER_DT_NUMBER]&uniqueCode=[UNIQUE_CODE]

Parameters:
meterNo = meter number
service = e.g IKEJA_TOKEN
phone = phone
amount = amount of electricity to purchase
meterType = e.g prepaid and postpaid OR as specified
customerName = Customer name from check API
customerNumber = Customer number from check API
customerAddress = Customer's Address (optional)
customerDtNumber = optional if provided it can be added
uniqueCode = this optional also but if provided can be added

Responses:
Success
{'status':201, 'message':'success', 'details': 'Transaction Successful'}
Failed
{'status':205, 'message':'failure', 'details': 'Transaction Not Successful'}
{'status':400, 'message':'error', 'details':'Insufficient Balance'}
{'status':400, 'message':'Bad Request', 'details':' Bad Request check your paramaters'}
End of Electricity API
